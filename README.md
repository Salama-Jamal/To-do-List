# To-do List

A simple browser-based to-do list app built with vanilla HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Save tasks to Firebase Realtime Database

## Development

1. Open `src/Index.html` in a browser.
2. Edit `src/main.js` and `src/styles.css` as needed.

## Files

- `src/Index.html` — app markup
- `src/styles.css` — styling
- `src/main.js` — app logic

## Notes

- Make sure the Firebase config in `src/main.js` is correct for your project.
